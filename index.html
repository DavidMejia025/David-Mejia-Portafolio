Jugar!!

Picas y fijas is a guesing game where one participant has the opportunity to find the secret number of four digits that the computer create. The game works as follow: The participant send four probable digits to the computer and the machine compute wheter the numbers are present (picas), in the same possition (fijas) or nowhere as the target number. In the next round the participant is informed the number of picas and fijas. This process continues until the guess number complete 4 fijas and the target number is discovered.


Tools: Javascript, Jquery, HTML, CSS.
